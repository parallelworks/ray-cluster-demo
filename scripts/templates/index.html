Submit via Ray Jobs API
RECOMMENDED

The Ray Jobs API is the standard way to submit work to a Ray cluster. It packages your script and dependencies, handles logging, and supports job management (status, stop, logs).

Install the Ray CLI on your local machine (if not already installed): 

uv pip install "ray[default]"   # or: pip install "ray[default]"

Open an SSH tunnel to the Ray dashboard: 

pw ssh -L 8265:localhost:8265 workspace

Then submit jobs from your local machine: 

# Submit a job
ray job submit --address http://127.0.0.1:8265 --working-dir . -- python my_script.py

# Check status and logs
ray job list --address http://127.0.0.1:8265
ray job logs --address http://127.0.0.1:8265 <job-id>
Example: my_script.py 
import ray
import socket
import time

ray.init()

@ray.remote
def compute(i):
    """A sample task that runs on a worker node."""
    time.sleep(1)
    host = socket.gethostname()
    return f"Task {i} completed on {host}"

# Submit 10 tasks across the cluster
futures = [compute.remote(i) for i in range(10)]
results = ray.get(futures)

for r in results:
    print(r)

Example: GPU hello world 
import ray
import socket

ray.init()

# Check available GPUs across the cluster
resources = ray.cluster_resources()
total_gpus = int(resources.get("GPU", 0))
print(f"Cluster has {total_gpus} GPU(s) across {len(ray.nodes())} node(s)")

@ray.remote(num_gpus=1)
def gpu_task(task_id):
    """Run a task on a GPU worker."""
    import os
    host = socket.gethostname()
    gpu_ids = os.environ.get("CUDA_VISIBLE_DEVICES", "none")

    # If torch is available, report GPU details
    try:
        import torch
        if torch.cuda.is_available():
            name = torch.cuda.get_device_name(0)
            mem = torch.cuda.get_device_properties(0).total_mem / 1e9
            # Quick matmul to verify GPU works
            a = torch.randn(1000, 1000, device="cuda")
            b = torch.randn(1000, 1000, device="cuda")
            torch.mm(a, b)
            return f"Task {task_id} on {host} GPU {gpu_ids}: {name} ({mem:.1f} GB) - matmul OK"
        return f"Task {task_id} on {host} GPU {gpu_ids}: torch found but CUDA unavailable"
    except ImportError:
        return f"Task {task_id} on {host} GPU {gpu_ids}: (install torch for GPU details)"

if total_gpus == 0:
    print("No GPUs in cluster. Request GPUs via the 'GPUs per Node' option.")
else:
    futures = [gpu_task.remote(i) for i in range(total_gpus)]
    for result in ray.get(futures):
        print(result)
Alternative: Connect via Python SDK

Forward the Ray GCS port and connect directly from Python: 

pw ssh -L 6379:localhost:6379 workspace
import ray

ray.init(address="ray://127.0.0.1:6379")

@ray.remote
def my_task(x):
    import time, socket
    time.sleep(1)
    return f"Result {x} from {socket.gethostname()}"

futures = [my_task.remote(i) for i in range(10)]
print(ray.get(futures))
ray.shutdown()
Alternative: Run Directly on Head Node

SSH into the head node and run scripts with Ray pre-configured. No tunnel needed. 

# SSH to the head node
pw ssh workspace

# Activate the Ray environment
source ~/pw/software/ray-*/bin/activate

# Set the Ray address
export RAY_ADDRESS="<RAY_HEAD_IP>:6379"

# Run your script
python my_script.py
